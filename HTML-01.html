Ferramentas utilizadas

(Voltar)

A construção do HTML e outros diversas linguagens de programação necessitam de certas ferramentas.

Algumas são bastante simples e outras um pouco mais complicadas.

Uma ferramenta de deselvolvimento web é algo que ajuda você a alcançar os objetivos do seu projeto de forma focada.

Para o curso apresentado pela DIO.me iremos utilizar a ferramenta VS Code, um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS.

Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Dentro dessa ferramenta iremos utilizar extensões como Live Server, uma extensão que te permite ver a sua página ao vivo no navegador.

Nos próximos tópicos iremos entender melhor essas ferramentas na prática.

Usando o Inspetor de Elementos

(Voltar)

Essa ferramente é diretamente disponibilizada pelo navegador chrome e nos permite visualizar os códigos HTML e CSS usados no desenvolvimento de uma página da web e copiar esses conteúdos para fazer alterações em ambientes externos.

Para utilizar você pode pressionar o botão f12 do teclado na página que deseja ispecionar ou as teclas Ctrl+Shift+I e passar o mouse sobre o elemento que deseja inspecionar.

Estrutura básica do HTML

(Voltar)

Agora passamos diretamente para o HTML em si, veremos sua estrutura e como ela funciona.

O código do HTML se divide em três partes:

"html" e "/html" sendo a primeira.

Vale ressaltar que todas as tags devem ser colocadas dentro dos sinais < >.

"head" e "/head" sendo a segunda dentro da primeira.
"body" e "/body" sendo a terceira abaixo da segunda e dentro da primeira.

A primeira tag "html" é utilizada para iniciar o código html que irá estuturar a página, como um recurso adicional do VS code, você pode digitar a tag "html:5" e apertar a tecla TAB do seu teclado o VS Code vai entender que você deseja iniciar a criação de um código HTML e irá preencher os campos internos com o restante da estrutura.

A segunda tag "head" contém as informações da página que ficaram a mostra na própria aba como um identificador do site, dentro dessa estrutura se utiliza a tag "title" e "/title" e seu conteúdo define o nome que será visível na aba.

A terceira tag "body" define o corpo da página, com titulos definidos pelas tags "h1", "h2", "h3", "h4", "h5" e "h6", cada número define o tamanho do titulo sendo o 1 o maior e o 6 o menor, também temos as tags "p" para parágrafos e "blockquote" para citações, mas iremos nos aprofundar mais no próximo tópico.

Essa é a definição básica de uma estrutura HTML.

Falando sobre tags

(Voltar)

Cada tag no código HTML tem uma função, tendo um início e um fim sinalizado pela "/" conforme informado no tópico anterior.

Elas são marcas que definem seus conteúdos, assim como explicado no tópico anterior temos a tag "p" que é utilizada no body do código para definir um parágrafro, temos a tag "blockquote" também utilizada no body para definir uma citação, temos também a tag "title" utilizada no head para definir o titulo da aba assim como a tag "h1" que define um título no body.

Temos outras tags com diversas funções para editar o texto como "strong" que deixa o texto em negrito, temos também a tag "i" que deixa o texto em itálico e também a tag "u" que sublinha o texto em questão.

Essas são apenas algumas das diversas tags que podemos utilizar no HTML, para uma pesquisa mais aprofundada você pode visitar o site W3Schools que contém as informações de todas as tags para HTML.

Atributos básicos

(Voltar)

É uma definição para a tag, podendo alterá-la ou complementar a mesma, muito utilizada no CSS.

Alguns exemplos que podemos utilizar é o atibuto "id" que pode ser atribuido a qualquer tag e serve como um identificador para a mesma.

Outro exemplo mais utilizado é dentro da tag "input", essa tag tem a função de permitir que o visitante do site digite algo em uma caixinha apresentada no site, o conteúdo da caixinha que será permitido a digitação pode ser controlado com o atributo da tag, podendo ser o atributo "text" que irá permitir a digitação de textos e o atributo "number" que irá permitir apenas a digitação de números, não sendo possível inserir letras.

Esses são apenas alguns exemplos de atributos também, vale ressaltar que cada tag pode oferecer diversos atributos dependendo da função daquela tag.

Executando uma página HTML

(Voltar)

Agora um pouco de prática!

Para executar uma página HTML devemos seguir os passos de todos os tópicos anteriores e criar o código da nossa página, crie da forma que desejar prestando atenção à estrutura do html presente no terceiro tópico e as tags presente no quarto tópico.

Após finalizar a criação do código ou durante a criação do mesmo, vá no explorador dentro do VS Code localizado na parte esquerda e encontre o arquivo HTML que você está editando, clique com o botão direito no arquivo e selecione a opção: "Open with Live Server" ou pressione as teclas "Alt+L Alt+O" que o seu arquivo irá abrir no navegador e você poderá ver o resultado do seu código.

Textos

(Voltar)

Os textos no código serão em sua maioria utilizados no "Body" do código onde iremos utilizar tags de titulos, os "h1"..., tags de parágrafos, o "p", tags de citações, o "blockquote".

Também utilizaremos as tags "strong", "i" e "u" para a formatação do nosso texto.

As combinações que podemos fazer com essas tags irá definir o nosso texto e sua apresentação.

Listas Ordenadas e Não Ordenadas

(Voltar)

A diferença entre essas listas é:

Lista Ordenada: Possuí uma ordem representada por números.

Lista Não Ordenada: Não possuí uma ordem e é representada por marcadores que podem ser alterados no CSS.

Para as listas Ordenadas iremos utilizar a tag "ol" para definir o início e fim da lista, vale ressaltar que para adicionar itens a essa lista é preciso utilizar a tag "li" pois sem ela o conteúdo não é mostrado.

Para as listas Não Ordenadas iremos utilizar a tag "ul" para definir o início e fim da lista, assim como na anterior, para adicionar itens é necessário utilizar a tag "li".

Links

(Voltar)

Essa função em nosso código nos permite navegar entre links disponibilizados na página, podendo ser sites terceiros ou partes do nosso próprio site.

Para utilizar essa função se deve utilizar a tag "a" ela representa função de links, para definir o site iremos utilizar o atributo "href=""" dentro desses dois pontos iremos colocar o link para que possamos ser redirecionados para o site, vale ressaltar que dentro da tag deve ser colocado o conteúdo que servirá de hiperlink, podendo ser uma palavra aleatória ou o nome do próprio site.

Dentro dessa tag temos outros atributos, um deles é o "target=""" dentro desses dois pontos temos as opções "_blank" que define o hiperlink a abrir em uma nova aba e a opção "_self" que define o hiperlink a abrir na mesma aba.

Outro atributo é o "title=""" dentro desses dois pontos você pode inserir o nome do site ou um texto aleatório e sempre que o mouse passar por cima do hiperlink sem clicar será mostrado uma caixa de texto com o texto que foi digitado nesse atributo.

Outra função interessante de hiperlink que podemos utilizar é a navegação na mesma página, onde o hiperlink vai apenas nos mandar para a parte da página com o conteúdo selecionado, para fazermos isso devemos definir na tag onde está o conteúdo uma "id" após definir essa id devemos ir na parte do texto onde iremos habilitar o hiperlink e adicionar normalmente a tag "a", no entanto ao invés de colocarmos um link no atributo "href=""" iremos colocar o símbolo "#" seguido do id do item.

Essas são algumas configurações que podemos utilizar ao habilitar hiperlinks no nosso código.
